# /dev - Запуск режима разработки

Запускает Vite dev сервер для локальной разработки.

## Инструкции

1. Проверь, что нет запущенного сервера на порту 5000
2. Запусти dev сервер:

```bash
npm run dev:client
```

3. Сервер будет доступен на http://localhost:5000

## Вывод

```
✅ Dev сервер запущен!

URL: http://localhost:5000

Hot reload включен - изменения применяются автоматически.
```

## Если порт занят

```bash
# Найти процесс
lsof -i :5000

# Убить процесс (если нужно)
kill -9 <PID>
```
